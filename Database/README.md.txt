# 데이터베이스

`데이터베이스`는 관련성을 가지며 중복이 없는 데이터들의 집합이다. 

이러한 데이터베이스를 관리하는 시스템을 `DBMS(데이터베이스 관리 시스템)`라고 부른다.

# MySQL

> RDBMS(관계형 DBMS)

```bash
$ mysql -h localhost -u root -p
```

* `-h ` 뒤에는 접속할 주소
* `-u` 뒤은 사용자 명
* `-p` 는 비밀번호를 사용하겠다.

> 프롬프트가 mysql> 로 바뀌면 정상적인 접속
>
> 나갈 떄는 exit
